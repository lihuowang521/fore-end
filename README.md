# fore-end
个人前端项目作品集（持续更新）
项目介绍
这份文档汇总了我开发的三个前端相关项目，涵盖静态展示网页、原生前端开发的毕业设计以及基于 Vue 框架结合 MQTT 协议的交互界面开发，完整体现了从基础前端开发到框架应用、协议集成的技术能力。
项目详情
1. neuro-sama 项目
项目概述
该项目是为Neuro-sama（全球顶流 AI VTuber） 开发的个人试水静态网页，核心用于展示 Neuro-sama 的相关信息，是前端静态页面开发的入门实践项目。
技术栈
HTML：构建页面基础结构
CSS：负责页面样式美化与布局
JavaScript（可选）：若包含简单交互逻辑，用于实现基础页面动效 / 事件响应
项目特点
纯静态页面，加载速度快，适配基础的信息展示需求
聚焦页面视觉呈现与内容排版，还原 AI VTuber 的品牌特征
无复杂交互逻辑，侧重前端基础语法的落地应用
2. 「前端」文件夹 - 智能压力传感器交互界面（毕业设计）
项目概述
此为个人毕业设计作品，纯使用前端三件套（HTML/CSS/JavaScript） 开发智能压力传感器的交互界面，完整实现传感器数据可视化、参数配置、状态监控等核心交互功能。
技术栈
HTML：搭建界面的 DOM 结构，划分数据展示区、操作区、告警区等模块
CSS：实现响应式布局，美化界面组件（图表容器、按钮、表单等），适配不同设备显示
JavaScript：处理传感器数据的逻辑计算、DOM 动态更新、交互事件（如参数修改、数据刷新）
项目特点
无框架依赖，纯原生前端开发，深度掌握前端基础原理
聚焦传感器数据的可视化展示（如数值实时更新、趋势图表绘制）
实现人机交互核心逻辑，如参数配置提交、异常数据告警提示
3. dist 文件夹 - Vue+MQTT 智能压力传感器交互界面
项目概述
基于 Vue 框架重构的智能压力传感器交互界面，核心优化了代码架构与可维护性，并集成MQTT 协议实现与传感器设备的实时通讯，是对毕业设计的技术升级版本。
技术栈
Vue 框架：采用组件化开发思想，拆分界面模块（如数据卡片、配置表单、图表组件），提升代码复用性
HTML/CSS/JavaScript：基础前端语法结合 Vue 语法实现界面开发
MQTT 协议：集成 MQTT 客户端库，实现传感器数据的实时订阅 / 发布，完成前端与设备的双向通讯
项目特点
组件化开发，代码结构清晰，便于后期维护与功能扩展
基于 MQTT 协议的实时通讯，精准、高效获取传感器数据
结合 Vue 的响应式特性，实现数据与界面视图的实时联动
